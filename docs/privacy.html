Privacy Policy

Last updated: June 15, 2026

X Non-Follower Checker / X 未回关检测 is a Chrome extension that helps users scan their X / Twitter following list and identify accounts that do not show the "Follows you" marker.

Data Collection

The extension does not collect, transmit, sell, or share user data. Account handles, following list entries, scan results, and batch unfollow progress are processed locally in the user's browser.

Data Storage

The extension may store temporary scan state in Chrome session storage so the popup or side panel can display progress and results. This data remains local to the user's browser and is not uploaded to any external server.

Permissions

  • activeTab: Used to access the current X / Twitter tab after the user starts a scan.
  • scripting: Used to inject packaged local scripts into X / Twitter pages.
  • tabs: Used to navigate the current tab to the user's following list and open selected profiles.
  • storage: Used to store local session state in the user's browser.
  • sidePanel: Used to display scan results in Chrome's side panel.
  • Host permissions for x.com and twitter.com: Required for the extension to run on X / Twitter pages.

Remote Code

The extension does not use remotely hosted code. All JavaScript and CSS are packaged with the extension.

Third Parties

The extension does not share user data with third parties and does not use user data for advertising, creditworthiness, or unrelated purposes.
